# 📘 Assignment: Java Text Processing

## 🎯 Objective

Master string manipulation, file input/output operations, and text processing techniques in Java by building utilities that read, analyze, and transform text data.

## 📝 Tasks

### 🛠️ String Manipulation and Analysis

#### Description
Write Java methods that perform various string operations including character counting, substring extraction, and case conversion. Create a `TextAnalyzer` class that processes and analyzes text content.

#### Requirements
Completed program should:

- Implement methods to count character frequencies in a string
- Convert text to uppercase and lowercase formats
- Find and replace substrings within a text
- Validate input strings and handle edge cases gracefully

### 🛠️ File Input/Output Operations

#### Description
Develop a program that reads text from files, processes the content, and writes the results to new files using Java's file I/O APIs.

#### Requirements
Completed program should:

- Read content from an input text file
- Process the file content (e.g., filter, transform, or analyze)
- Write processed results to an output file
- Handle file not found and I/O exceptions appropriately

### 🛠️ Text Transformation Utility

#### Description
Create a comprehensive text processing utility that combines string manipulation and file I/O to transform text documents (e.g., word counting, line numbering, or format conversion).

#### Requirements
Completed program should:

- Accept input text from a file
- Perform text transformations (remove duplicates, sort lines, count words/lines)
- Format the output in a readable way
- Save results to an output file with proper formatting
